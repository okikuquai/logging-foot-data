## Overview
This project acquires the values ​​of the following sensors and saves them in csv.

* MBX055
    * Accelaration Sensor
        * X-axis
        * Y-axis
        * Z-axis
    * Gyro Sensor
        * X-axis
        * Y-axis
        * Z-axis
* MCP3425(ADC for Pressior Sensor: MF02A-N-221-A01)

## Required
* Python 3.9.2
* Raspberry pi Zero
* I2C device needs to be connected to channel 1
